SWIFT-TEC v3

Global TEC Forecast (3-day, 加算モデル)
・BoM TEC（最大9枚）から 3日 / 30min 予報を生成
・BaseTEC（静穏）を「入力TEC − Sstorm_base − Sflare_base」から逆算
・Base に昼夜ウェイト（夜間減衰）を適用して横長ノッペリを抑制
・NOAAテキスト / Kpテーブルから Kp(t) を自動読取
・Leaflet 地図で TEC / GPS誤差を確認、地域フォーカスも可能

2. NOAA Kp テキスト（予報用）

3-Day Forecast を丸ごと貼付
・Kp（3日予報）：
　→ 3-Day Forecast をそのまま貼り付け
形式が合わない場合は、簡易フォーマット：
YYYY-MM-DD HH Kp
に整形すると確実です。

2b. Base用 Kp シリーズ

Base 抽出用 Kp(time) テーブル
・例：NOAAの観測Kpを JSON 配列で貼り付け
　[["time_tag","Kp","a_running","station_count"],
　 ["2025-11-26 00:00:00.000","3.33",...], ...]
・空欄の場合は 2. の Kp(t) を Base 用にも流用します。
