・入力TECは BoM or NOAA(GloTEC) を選択

8b. GNSS / DOP設定

multi-GNSS / satellite select
・GPS / Galileo / GLONASS / BeiDou / QZSS のTLEを読み込み、選択衛星だけでDOPを計算します。
・各衛星は 使用ON/OFF と Active/Inactive を手動切替できます。Inactive衛星はDOP計算から除外します。
・DOPモードでは、GDOP / PDOP / HDOP / VDOP / TDOP と DOP×TEC誤差を表示できます。
・TEC表示はNOAA 30分値をそのまま矩形描画します。0補完・フーリエ平滑化・自動再スケールは行いません。
